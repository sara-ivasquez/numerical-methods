# NuméricaLab

Plataforma interactiva para resolver métodos numéricos.

## Métodos implementados
- **Búsqueda de Raíces:** Bisección, Regula Falsi, Punto Fijo, Newton-Raphson, Secante, Raíces Múltiples 1 y 2
- **Sistemas de Ecuaciones:** Jacobi, Gauss-Seidel, SOR
- **Interpolación:** Vandermonde, Newton, Lagrange, Spline Lineal, Spline Cúbico

## Tecnologías
React 18 + Vite + React Router + Math.js + Recharts

## Instalación

```bash
npm install
npm run dev
```

Abre http://localhost:5173

## Deploy
Conecta el repositorio a [Vercel](https://vercel.com) y despliega automáticamente.
